# 🌤️ Weather App

A simple and responsive Weather Application that allows users to check real-time weather conditions of any city or village using the OpenWeather API.

---

## 🚀 Live Demo

🔗 https://shivank302.github.io/weather-app-project/

---

## 📌 Features

* 🌍 Get weather using current location (GPS support)
* 🔎 Search weather by city or village name
* 🌡️ Real-time temperature in Celsius
* 💧 Humidity, Wind Speed & Pressure details
* 🌄 Dynamic background based on weather condition
* 📱 Fully responsive design
* 🎬 Splash screen animation

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* OpenWeather API

## ⚙️ How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/shivank302/weather-app-project.git
```

2. Open the project folder

3. Run `index.html` in your browser
